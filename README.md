# Full Stack Open Course Solutions

This repository hosts my personal solutions while progressing through the University of Helsinki's Full Stack Open curriculum. The course combines modern JavaScript tooling with an end-to-end look at building production-ready single page applications.

## Course Snapshot

- Focus: React-driven SPA development backend by RESTful Node.js APIs
- Tech stack: React, Redux, Node.js, Express, MongoDB, GraphQL, TypeScript, Jest, Cypress, and related tooling
- Outcomes: Strong foundations in frontend state management, backend API design, database modeling, testing, and continuous integration
